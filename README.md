# Proyecto - Tecnologías Web

## Para iniciar el proyecto, se deben ejecutar los siguientes comandos:

- `npm install`
- `ng serve`

## Una vez ejecutados los comandos, se debe ingresar al enlace:

## http://localhost:4200
